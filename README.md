## Installation

(TBA)

### From file

Download the latest version of plasmoid from [KDE Store][kdestore] (TBA)

#### A) Plasma UI

1. Right click on panel or desktop
2. Select *Add Widgets > Get New Widgets > Install From Local File*
3. Choose downloaded plasmoid file

#### B) Terminal

```sh
kpackagetool5 -i plasma-simplyplural-*.plasmoid
```

### C} From GitHub

Clone repository and go to the project directory

```sh
git clone https://github.com/EndlessEden/plasma-simplyplural.git
cd plasma-simplyplural/package
kpackagetool5 -i .
```
